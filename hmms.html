Key Features

Meal Scheduling

Easily create and manage meal schedules for the entire semester with our intuitive interface.

Different access levels for students, staff, tutors, and hall provost with appropriate permissions.

Real-time Analytics

Get insights into meal consumption patterns, preferences, and financial reports.

Payment Integration

Seamless payment processing for meal plans with multiple payment options.

Mobile Friendly

Role-based Features

See how HMMS works for different user roles in our hall community

Student Portal

Features for Students:

  • View and select daily meal
  • Track meal credits and balance
  • Submit special requests if forgot meal on/off
  • View meal calendar and announcements
  • Give feedback and ratings for meal
House Tutor Portal

Features for House Tutors:

  • Approve student meal requests
  • Monitor student meal participation
  • Generate reports hall meal cost
  • Communicate with hall provost
  • View student meal cost
Staff Portal

Features for Staff:

  • Manage daily meal preparation
  • Track inventory and supplies
  • View everyday student meal on/off list
  • Create Impressive Meal Plan
  • Add Daily, Month Meal Cost
Hall Provost Portal

Features for Hall Provost:

  • Overall meal plan management
  • Financial oversight and reporting
  • Approval of special requests
  • System configuration
  • User management and permissions
